A full-stack web application that helps users organize their clothes and generate outfit recommendations.

---

## Tech Stack

- FastAPI (Backend)
- React (Frontend)
- PostgreSQL (Database)

---

## Features

- User signup and login
- Clothing item management
- Outfit recommendations

---

## How to Run the Project

### Backend (FastAPI)

Navigate to the backend folder and run:
uvicorn app.main:app --reload


---

### Frontend (React)

Navigate to the frontend folder and run:


npm install
npm start
