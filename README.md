# 👕 Digital Wardrobe Organizer

A full-stack web application that helps users manage their wardrobe, track clothing usage, and generate outfit recommendations based on conditions like weather.

---

## 🚀 Tech Stack

**Frontend**
- React
- Tailwind CSS (ShadCN UI)

**Backend**
- FastAPI (Python)

**Database**
- PostgreSQL

---

## ✨ Features

- 🔐 User authentication (Signup / Login)
- 👕 Upload and manage clothing items
- 🧠 Smart outfit recommendations
- 🌦️ Weather-based outfit suggestions
- 🧺 Laundry tracking system
- ✔️ Track worn items
- 💾 Save outfits for later
